## 🎯 Visão Geral
Este repositório é dedicado ao desenvolvimento de um estudo focado em como a Ciência de Dados pode ser uma ferramenta de transformação social. O objetivo central é analisar o papel das instituições financeiras na promoção de uma economia mais justa e acessível, utilizando a tecnologia para democratizar o acesso ao crédito.

## 🏢 A Empresa: Nubank
O Nubank foi selecionado como objeto de estudo por sua abordagem inovadora e missão de simplificar a experiência bancária. No campo da Ciência de Dados, a empresa se destaca por empregar algoritmos e Big Data para embasar decisões que vão desde a segurança nas transações até a personalização do atendimento.

## 🌍 Ligação com os Objetivos de Desenvolvimento Sustentável (ODS)
O projeto está diretamente alinhado com o **ODS 10 – Redução das Desigualdades**. 
Acreditamos que a democratização do acesso a serviços bancários é um passo fundamental para reduzir a disparidade social. O estudo explora como a análise inteligente de dados permite identificar perfis de crédito de forma mais inclusiva, integrando pessoas que historicamente foram excluídas do sistema tradicional.

## 📈 Propósito do Estudo
O objetivo é transformar dados em impacto real através de:
1. **Análise de Variáveis:** Compreender os indicadores socioeconômicos que influenciam o risco de crédito.
2. **Justiça na Avaliação:** Discutir como modelos preditivos podem tornar a concessão de crédito mais justa e minimizar a exclusão financeira.
3. **Pensamento Computacional:** Aplicar pilares de abstração e análise de padrões para prever a saúde financeira dos clientes.

## 📁 Estrutura do Repositório
Para facilitar a navegação e atender aos critérios de organização, o repositório está dividido em:

* **`documentos/`**: Relatórios técnicos e o PDF da Entrega 1.
